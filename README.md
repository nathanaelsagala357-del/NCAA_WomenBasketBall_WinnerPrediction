# DSCI 100 FINAL PROJECT

Data Analysis prediction regarding the NCAA Women Basketball, March Madness. The prediction are made using k-nearest neighbours algorithm to classify the final team's status, using the winner status as the predictors and the seed ranks, conference wins percentage and total wins percentage as predictors variable. The model will be run first with cross validation with 5 folds to determine the optimal number of neighbours and crosschecked with neighbours from k = 1 to k = 20.
